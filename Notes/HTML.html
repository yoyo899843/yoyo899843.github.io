HTML

基礎結構(HTML5)


<!DOCTYPE html>
<html lang="zh-TW">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>titleg</title>
    <link rel="stylesheet" href="Style.css">
</head>
<body>
    <header>

    </header>
    <section>

    </section>
    <footer>

    </footer>
</body>
</html>
  • <header>

網頁最上方的部分,通常放導覽列及標題

  • <section>

網頁中間的各個區塊,預設是垂直排列

  • <footer>

網頁最下方的部分,通常放版權宣告

常用

  • class

用於標註該區段名稱，以便CSS辨認


<section class="style"></section>
.style{background-color=white} //CSS裡自訂該區背景色
  • <h1></h1>

標題，還有h2、h3、h4....

  • <p></p>

內文

  • <a href="example.html">

連結

  • <img src="example.jpg" alt="pic title">

顯示圖片，alt是盲人使用朗讀模式會讀出來的內容

  • <ul></ul>

列表，開頭為點，搭配<li></li>使用

  • <table></table>

表格，內部結構為


<table>
    <thead> //表格標題
        <h1></h1>
    </thead>
    <tbody> //表格內容
        <tr> //橫排由左至右
            <td> //橫排中第一格
            </td>
        </tr>
        <tr>
            <th> //橫排中第一格,並預設粗體當作標題
            </td>
        </tr>
    </tbody>
</table>
  • <div></div>

把裡面的所有區塊合成一個區域
