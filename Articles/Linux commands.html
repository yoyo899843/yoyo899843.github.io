Kali 常用

工具

  • net tools

更新/安裝

  • sudo apt-get update 用來取得遠端更新伺服器的套件檔案清單。
  • sudo apt-get -y dist-upgrade 更新套件。
  • sudo apt-get clean 清除更新時所下載回來的更新(安裝)檔案。
  • sudo apt-get autoremove 自動清除更新後用不到的舊版本檔案（例如舊的核心程式）

檔案

  • find 尋找檔案
  • head 開啟檔案

網路

  • nmap
  • proxychain
  • ifconfig

破解

  • cupp 字典產生器
  • crunch 字典產生器
  • burpsuit 暴力破解

弱點掃描

  • nessus
